# Fetal Health Classification App

This project is a Python-based desktop application that predicts fetal health using Cardiotocography (CTG) data. The system uses machine learning models such as **Random Forest** and **Logistic Regression** to classify fetal conditions into **Normal, Suspect, or Pathological**. The application includes an interactive **Tkinter GUI** where users can input 21 CTG parameters using sliders and instantly receive prediction results with probability scores. It also provides **EDA visualizations**, **feature importance**, and **model performance metrics** like confusion matrix and accuracy. Built using **Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn**, this project demonstrates applied statistical analysis and machine learning for healthcare data classification.
